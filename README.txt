1、关于代码的几点说明：
（1）由于还未接入实际电路，因此过温保护程序暂时还未加入，连接实际电路后将prote.c文件的Check_Error（）函数注释解除即可

2、界面的相关说明：
（1）基本逻辑为坐标检索和切换——（x，y，z）代表一个界面对应的坐标，根据不同的坐标显示不同的界面，按键动作主要就是更新坐标位置
（2）光标闪烁的说明——光标为倒三角光标，是自行创建的图标，其对应的字库放在Front.h下Pointleft&Pointright

3、关于LCD驱动的说明：
（1）主要的字符接口：Write_Single_16x32AsicII（）、Write_String_16x32AsicII（）、Write_Single_12x24AsicII（）、Write_String_12x24AsicII（）、Write_Single_8x16AsicII（）、Write_String_8x16AsicII（）
接口的介绍参见spi_oled.c文件
（2）接口的主要参数：行（0-63）、列（0-63），其界面坐标图参见界面设计excel文件

（1）主要的两个接口：SPI_AT25_Read（）、SPI_AT25_Write（），基于这两个接口可以读/写对应地址的内容，后续的拓展开发可以围绕这两个接口展开
